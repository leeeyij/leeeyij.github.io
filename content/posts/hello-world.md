## 欢迎来到我的博客

这是我的第一篇博客文章！🎉

### 技术栈

这个博客使用以下技术搭建：

- **[Hugo](https://gohugo.io/)** - 世界上最快的静态网站生成器
- **[PaperMod](https://github.com/adityatelange/hugo-PaperMod)** - 简洁、快速、响应式的 Hugo 主题
- **GitHub Pages** - 免费的静态网站托管

### 特性

- ✅ 响应式设计，支持移动端
- ✅ 深色/浅色模式自动切换
- ✅ 内置搜索功能
- ✅ 文章归档页面
- ✅ 标签和分类系统
- ✅ 阅读时间估算
- ✅ 代码复制按钮
- ✅ RSS 订阅

### 下一步

1. 修改 `hugo.yaml` 中的站点配置（标题、作者、社交链接等）
2. 替换 `baseURL` 为你的 GitHub Pages 地址
3. 在 `content/posts/` 目录下创建更多文章
4. 运行 `hugo server -D` 本地预览
5. 推送到 GitHub 仓库启用 GitHub Pages

感谢访问！
